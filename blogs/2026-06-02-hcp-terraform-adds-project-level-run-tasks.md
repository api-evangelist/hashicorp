HCP Terraform now supports project-level run tasks, enabling organizations to apply security, compliance, and operational guardrails consistently across groups of workspaces while reducing manual overhead. Governance controls cascade automatically to all workspaces within a project, eliminating workspace-by-workspace configuration and ensuring new workspaces instantly meet required controls.
